TÂCHE : Implémenter du walk-forward retraining qui réentraîne le modèle 
glissant pour absorber les changements de régime.

CONCEPT :
- Fenêtre d'entraînement glissante de 3 ans
- Réentraînement tous les 3 mois  
- Test OOS sur les 3 mois suivants
- Embargo purge_hours entre train et OOS conservé

PLAN TECHNIQUE :

   `walk_forward_train(df, model_factory, train_months=36, step_months=3,

   Générateur yieldant (modèle, période train, période test) à chaque fold.

2. Ajouter dans learning_machine_learning/pipelines/base.py :
   `run_walk_forward(self) -> tuple[pd.DataFrame, dict]` qui :
   - Itère sur les folds via walk_forward_train
   - Agrège les prédictions OOS dans une seule série temporelle
   - Applique le backtest sur les prédictions agrégées
   - Retourne (predictions_aggregated, metrics_per_fold + aggregated)
